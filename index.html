VPN Debug Helper

Generate copy-ready commands for VPN traffic captures, debug log collection, and file transfers. Select your gateway version and flow details above — commands update automatically.

Follow this workflow to debug VPN issues:

  1. Flow Details: Enter the gateway IPs and optional local/remote hosts
  2. Debug Commands: Use the "VPN Debugging" tab to enable logs and collect debug data
  3. Captures: Run traffic captures in the "Traffic Captures" tab to analyze packets
  4. Collection: Package logs and captures into a tarball using "File Collection"
  5. Transfer: Upload files to an SFTP server using the "Transfer" tab
💡 Tip: All commands are copy-ready. Commands update automatically as you fill in fields. Everything happens locally — no data is sent anywhere.
